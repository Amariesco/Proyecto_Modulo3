Ejercicios del Bootcamp - Módulo 3

Se solicita para el proyecto:

------------------------------------------------------
Lección 6: Funciones en Python

 Objetivo: Modularizar todo el sistema dividiendo la lógica en funciones
reutilizables, algunas con retorno y parámetros.

 Tareas a desarrollar:
 Reorganizar el sistema en funciones con parámetros y return.
 Diferenciar variables locales y globales en funciones.
 Crear procedimientos para impresión, menú y validaciones.

 Archivos esperados:
 /modulos/funciones_utiles.py (todas las funciones generales)
 /main.py (llamado estructurado a funciones)

-----------------------------------------------------

**Los archivos:
 /Modulos/__init__.py (Para que python trate el direcctorio como paquete)
 /Modulos/sator_basicos.py (Para solicitar por pantalla los datos, en las variables nombre, edad, tipo de usuario, etc. y retornarlos al main)
 /Modulos/menu.py (Menu que se visualiza, se agrega eliminacion de usuario)
 /Modulos/gestion_datos.py (Base de datos y operaciones logicas, como imprime los registros de usuarios, como busca usuario por nombre y como elimina usuario por nombre)
 /Modulos/funciones_utiles.py (Herramientas generales del sistema, distincion entre variables locales y globales)
 main.py (Archivo central)
 README.md (Para dejar las notas necesarias para entender el proyecto)
