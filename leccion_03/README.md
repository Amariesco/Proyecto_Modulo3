Ejercicios del Bootcamp - Módulo 3

Se solicita para el proyecto:

------------------------------------------------------
Lección 3: Sentencias condicionales

Objetivo: Aplicar lógica condicional para tomar decisiones según los datos
ingresados.

Tareas a desarrollar:
 Evaluar reglas como “si edad > 60, categoría = adulto mayor”.
 Implementar condiciones múltiples (if, elif, else).
 Gestionar errores de entrada con validaciones condicionales.
 Usar expresiones booleanas y lógica anidada.

Archivos esperados:
 /modulos/validaciones.py (funciones de validación)
 /main.py (invocación de las validaciones)
-----------------------------------------------------

**Los archivos solicitados para el proyecto:
 /Modulos/__init__.py (Para que python trate el direcctorio como paquete)
 /Modulos/datos_basicos.py (Para solicitar por pantalla los datos, en las variables nombre, edad, tipo de usuario, etc. y retornarlos al main)
 /Modulos//validaciones.py (valida a que categoria corresponde la edad ingresada, si la nota ingresada corresponde al rango hasta 7.0 max, y valida en que categoria esta el estado del alumno)
 main.py (Archivo central, Codigo basico en main.py para calcular operaciones basicas con los datos solicitados en datos_basicos.py e imprime los resutados por pantalla, se agregan invocacion a las validaciones)
 README.md (Para dejar las notas necesarias para entender el proyecto)
