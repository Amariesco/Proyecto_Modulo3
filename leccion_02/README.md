Se solicita para el proyecto:

------------------------------------------------------
Lección 2: Tipos de datos y sentencias básicas
Objetivo: Utilizar variables, tipos de datos y operadores para recibir y
almacenar información del usuario.

Tareas a desarrollar:
● Declarar variables para capturar nombre, edad, tipo de usuario, etc.
● Validar tipo de dato con int(), float(), str(), etc.
● Utilizar operadores aritméticos, de comparación y lógicos.
● Crear los primeros registros usando listas y diccionarios.

Archivos esperados:
● /main.py (actualizado)
● /modulos/datos_basicos.py (funciones de carga de datos)

-----------------------------------------------------

**Se crean los archivos necesarios para el proyecto:
/Modulos/__init__.py (Para que python trate el direcctorio como paquete)
/Modulos/sator_basicos.py (Para solicitar por pantalla los datos, en las variables nombre, edad, tipo de usuario, etc. y retornarlos al main)
main.py (Archivo central, Codigo basico en main.py para calcular operaciones basicas con los datos solicitados en datos_basicos.py e imprime los resutados por pantalla)
README.md (Para dejar las notas necesarias para entender el proyecto)
